================================================================================
🗄️ DATABASE SETUP
================================================================================

PostgreSQL Setup:
  1. Install PostgreSQL: https://www.postgresql.org/download/windows/
  2. Create database: psql -U postgres -c "CREATE DATABASE finance_ai_dev;"
  3. Update DATABASE_URL in app/.env
  4. Run migrations: python -m alembic upgrade head

SQLite (Development):
  DATABASE_URL=sqlite+aiosqlite:///./finance_ai_dev.db

Migrations:
  cd app
  python -m alembic upgrade head        # Apply migrations
  python -m alembic downgrade base      # Reset database
  python -m alembic revision --autogen  # Create new migration

================================================================================
🔧 TROUBLESHOOTING
================================================================================

"ModuleNotFoundError: No module named 'src'":
  → Activate virtual environment: ..\venv\Scripts\Activate.ps1

"Database connection refused":
  → Check PostgreSQL is running
  → Verify DATABASE_URL in .env
  → Create database if missing

"Port 8000 already in use":
  → Kill process: netstat -ano | findstr :8000 then taskkill /PID <PID> /F
  → Or use different port in startup script

"Missing dependencies":
  → Reinstall: cd app && python -m pip install -e .
